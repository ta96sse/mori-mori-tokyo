# ブラウザ表示確認ワークフロー

ブラウザでの表示確認手順。

---

## 確認対象URL

| 環境 | URL |
|------|-----|
| 本番 | AGENTS.md のプロジェクト概要を参照 |
| ローカル | `yarn dev` 起動後のURL |

---

## 標準チェックリスト

### 1. 全体レイアウト
- [ ] ページが正常に表示されているか
- [ ] 背景色・フォントが正しいか
- [ ] ナビゲーションが機能しているか

### 2. 各セクション
- [ ] データが正しく表示されているか
- [ ] アニメーションが動作しているか
- [ ] リンクが機能しているか

### 3. レスポンシブ確認
- [ ] モバイル（375px）で崩れていないか
- [ ] タブレット（768px）で崩れていないか
- [ ] デスクトップ（1280px）で崩れていないか

### 4. パフォーマンス・アクセシビリティ
- [ ] 画像が表示されているか（broken imageがないか）
- [ ] フォントが正しく読み込まれているか

---

## 問題発見時のフロー

```
問題発見
  ↓
スクリーンショットで記録
  ↓
問題を分類:
  ├── データの問題 → データソースを確認
  ├── CSS/コンポーネントの問題 → ワーカーへ委任（delegate.md）
  └── ビルドの問題 → ビルドコマンドで確認
```
